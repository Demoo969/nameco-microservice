# nameco-microservice

  Its simply micro service code
  
 
To run project use: 
```shell
1 $ python3 -m venv env
2 $ pip install nameko in 
3 $ service folder/nameko run [name of service]
```
To use project:
```shell
$ python3 main.py -m/--method [-t/--text = 'text message'] [-n/--number = 1] 
```
